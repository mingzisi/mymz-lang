# 没有名字(mymz)编程语言

**本语言仅支持64位系统（x86-64、ARM64、RISC-V 64）。指针宽度、内存布局均假定为64位。**

**需要确保指针只使用低59位**

## 设计

- **一切皆值**：类型和函数是值
- **宏**：语法都由宏定义，原始语法中甚至**没有循环**
- **零开销抽象**：编译时确定的部分**不会带来运行时开销**

## 开发进度

**设计已完成，编译器开发中**

(语法和值编码有可能会改变)

## 基本语法

使用`。`结束一条语句。

`{}`表示代码块，用来管理作用域。

`//`后到换行符之间的内容是**注释**，不会被执行。

`/*`和`*/`之间也是**注释**，可跨行。

## 变量和常量

```mymz
定义 整数 变量 甲。
定义 整数 常量 乙:=0。
```

用`定义`定义一个变量或常量。如果要定义多个变量，将它们用`，`隔开。

声明变量时后面可以直接加`:=`进行初始化。如果这么做，可以不指定类型（会自动改为和值相同的类型）。

常量**必须**在声明时初始化。

```mymz
定义 整数 变量 甲，常量 乙，字符 丙，浮点数 变量 丁。
/*
甲：整数变量
乙：整数常量
丙：字符常量
丁：浮点数变量
*/
```
在类型后加`[元素个数]`将类型改为**数组**。**数组**可以同时存储多个类型相同的元素。数组使用`名称[索引]`访问元素。**索引从0开始**。

使用`的 长度`获取长度。它是一个**无符号整数**。

```mymz
定义 整数[10] 变量 数组。
定义 无符号整数 长度:=数组 的 长度。//10
```

支持自动推导类型：

```mymz
定义 常量 乙:=甲。//乙和甲类型相同，但是常量
定义 变量 乙:=甲。//同理
```
### 字面量

#### 数

直接写，可以用科学记数法：

```mymz
定义 整数 常量 零:=0。
定义 浮点数 常量 π:=3.1416。
定义 整数 常量 一亿:=1e8。//或1E8
```

**不区分大小写。**

#### 字符和字符串

字符串是**字符数组**。

非ASCII字符只能用字符数组存储，需要占多个字符。

字符用`‘’`，字符串用`“”`。

字符串可以跨行，`“`、`”`和`\`需要转义。

```mymz
定义 字符[9] 变量 字符串:=“a string”。
定义 字符[] /*自动判断长度*/ 变量 引号:=“\“\””。
```
### 作用域

变量的**作用域**为**所在代码块**，离开时将会被**自动删除**。

允许声明同名变量，但必须**不在同一作用域中**。如果声明名称相同的变量，之前的同名变量将会**隐藏**，直到新声明的变量离开作用域被删除后才重新可见。

## 指针和引用

### 指针

在类型后加`指针`声明一个**指针**，它可以指向对应类型的元素。使用`&`获取变量的**地址**。指针可以指向指针。用`*`解引用，获取指针指向的引用。

```mymz
定义 整数 变量 值。
定义 整数 变量 指针 变量 甲=&值。
定义 整数 变量 指针 常量 乙=&值。
定义 整数 常量 指针 变量 丙=&值。
```

甲的类型是指向整数变量的指针变量，既可以被修改，也可以修改指向的值；乙不能被修改，但可以修改指向的值；丙可以被修改，但不能修改指向的值。

指向类型为变量的指针可以指向常量，但反过来不行。

可以使用`分配`关键字分配存储空间。

```mymz
定义 整数 变量 指针 变量 甲:=分配 整数。
*甲:=0。
*甲+=1。
释放 甲。
```

不要释放不是用`分配`分配的存储空间，也不要多次释放同一片存储空间。可以对空指针使用释放。

**当然，也不要不释放空间。**

可以但**不推荐**使用无类型指针。

### 引用

```mymz
定义 整数 变量 甲。
定义 整数 变量 引用 乙:=甲。
```

这会声明一个**引用**，甲和乙完全相同。
乙被修改时甲的值也会变化。

```mymz
定义 常量 引用 乙:=甲。
```

这次不能用乙修改甲。

引用变量的作用域比源变量大时会导致**悬垂引用**，这是**未定义行为**。

### 停用名称和移动语义

使用`停用`停止使用一个名称。

```mymz
定义 整数 变量 甲。
甲:=0。
定义 整数 变量 引用 甲的引用。
停用 甲。
定义 浮点数 变量 甲。
甲:=0.1。//甲是浮点数变量
甲的引用:=1。
```

停用不影响作用域，使用引用仍然可以修改原变量，不会导致悬垂引用。

```mymz
定义 整数[1000] 变量 很大。
定义 整数[1000] 变量 移动。
移动:=很大。
停用 很大。
{
	定义 整数[1000] 变量 局部。
	移动:=局部。//也会触发移动
}
```

当变量被赋值后值源变量就失效时，将会触发移动语义。

## 复合类型

```mymz
定义 类型 常量 二维向量:=组合[浮点数 x，浮点数 y]。

二维向量 A。
A 的 x:=0。
A 的 y:=0。
```

类型也是值，可以用常量存储。用`的 类型`获取类型。

**类型不存储可变性。**

用`组合`定义的类型可以**同时存储所有成员**。

可以用`类型`类型的变量作为类型声明其它变量。

使用未初始化的类型变量声明其它变量是**未定义行为**。

## 程序流控制

只有跳转和条件跳转，但在后面会看到可以用**宏**写出其它的控制流。

### 跳转

用`定义 跳转 <名称>`声明一个**跳转点**。

`跳转 <名称>`将**跳转**到那里。

```mymz
定义 无符号整数 变量 计数器:=0。
定义 跳转 循环开始。
//……
计数器:=计数器+1。
如果 计数器<10 跳转 循环开始。
```

### 条件跳转

```mymz
定义 字节 变量 条件。
条件:=0。
如果 条件 跳转 条件满足。
//……
定义 跳转 条件满足。
```

## 运算

`:=`：赋值(前面已经出现过很多次了)

`()`在括号中的将会**优先**计算

`+-×÷`：数学运算，加减(相反数)乘除。

`mod`：取余数

`甲:=甲+乙`可以写成`甲+=乙`，其它运算也是一样的

`=≠＞＜≥ ≤`：比较

`& | ~ ^ << >>`：位运算

除了`+-`和`×÷`，其它运算混合使用都是未定义行为。编译时会报错，需要加`()`。

## 函数

函数也是值。

```mymz
定义 函数[整数，整数。返回 整数] 常量 加:
=函数[整数 甲，整数 乙。返回 整数]{
	返回 甲+乙。
}
定义 整数 常量 一加一的和:=和(1，1)。
```

`[]`中是**参数**和**返回类型**，后面中是函数要执行的语句。返回类型不写就是无返回值。

使用`返回`关键字让函数返回一个值。类型不符合时会**自动强制转换**。

返回导致**临时变量失效**也可以触发**移动语义**。

传入的参数时将使用**值传递**，修改不影响远变量。如果要在函数中修改变量的值，需要使用引用。

变量可以被传给接受常量引用的函数，常量不能传给接受变量引用的函数。

所有代码块都是无参数和返回值的函数类型，可以自动转换。

函数被传递到内部变量失效的地方并调用会导致**悬垂引用**，同样是**未定义行为**。

函数最多只能有65535个参数。

## 命名空间

命名空间可以隔离名称，你可以在不同的命名空间里使用相同的名称而不会引起歧义。

用`外部：`在命名空间中指定使用外部的名称。命名空间的名称不能是`外部`。

```mymz
定义 命名空间 示例命名空间。
定义 整数 变量 甲。
命名空间 示例命名空间{
	定义 整数 变量 甲:=0，乙。
}
甲:=示例命名空间：甲。 //将外部的甲赋值为内部的甲
使用 命名空间 示例命名空间。//使用整个命名空间
乙:=甲。//这是示例命名空间内的甲
外部：甲+=1。//外部的甲
停用 示例命名空间。
甲:=1。//外部的甲
使用 示例命名空间：乙。//使用示例命名空间内的乙
乙+=甲。
```

命名空间可以嵌套。

## 宏
用`定义 []:=[]。`定义一个宏。当某段代码和前面`[]`中的内容匹配时，它将被**替换**成后面`[]`中的内容。
使用`(类型 名称)`可以捕获定义的变量。
匹配`[ ] ( )`时需要用`\`**转义**。

**示例：**

```mymz
定义 类型 常量 二维向量:=组合[浮点数 x，浮点数 y]。
定义 函数[浮点数，浮点数。返回 二维向量] 常量 构造二维向量:
=函数[浮点数 x，浮点数 y。返回 二维向量]{
	定义 二维向量 结果。
	结果 的 x:=x。
	结果 的 y:=y。
	返回 结果。
}
定义 [\((浮点数 常量 x),(浮点数 常量 y)\)]:=[构造二维向量(x，y)]。

定义 函数[二维向量 常量 引用,二维向量 常量 引用。返回 二维向量] 加:
=函数[二维向量 常量 引用 甲,二维向量 常量 引用 乙。返回 二维向量]{
	定义 二维向量 结果。
	结果 的 x:=甲 的 x+乙 的 x。
	结果 的 y:=甲 的 y+乙 的 y。
	返回 结果。
}
定义 [(二维向量 甲)+(二维向量 乙)]:=[加(甲，乙)]。

定义 二维向量 A:=(1,1)+(2,2)。//可以直接这样
```

可以用`停用 []。`来停止对应匹配器的宏。

`停用`只能在被停用的宏的作用域中使用。

宏的作用域和变量相同，离开作用域后失效。

前面提到过，代码块可以自动转换成函数。所以可以这么做：

用`定义 标志符`在宏内创建一个**唯一**的标志符。它不会和外部的名称冲突。

```mymz
定义 [循环 (无符号整数 次数) 次 (函数[] 循环体)]:
=[
	定义 标志符 循环开始。
	定义 跳转 循环开始。
	定义 标志符 计数器。
	定义 无符号整数 变量 计数器:=0。
	如果 计数器＜次数 跳转 循环开始。
]

定义 整数 变量 甲:=2。
循环 5 次 {
	甲×=2。
	甲-=1。
}
```

`[]`将其中的匹配器合并为整体，合并后无法进行运算，只能原样展开。

用`([]……)`匹配任意次(包括0)重复：

```mymz
定义 函数[整数，整数。返回 整数] 常量 最大值:
=函数[整数 甲，整数 乙。返回 整数]{
	如果 乙＞甲 跳转 乙更大。
	返回 甲。
	定义 跳转 乙更大。
	返回 乙。
}

定义 [最大值\((整数 第一个)，([[(整数)，]…… (整数)] 其它)\)]:
=[最大值(第一个，最大值(其它))]。
```

### 编译时运行

用`编译时`标记代码在编译时运行。

```mymz
定义 字节 常量 使用英语:=1。
编译时{
	如果 ~使用英语 跳转 不使用英语。
	定义 [define]:=[定义]。
	定义 [variable]:=[变量]。
	定义 [int]:=[整数]。
	定义 [;]:=[。]。
	停用 定义。
	停用 变量。
	定义 跳转 不使用英语。
}

define int variable a;
```

在`编译时`块内部的代码只能访问**常量**或**编译时定义的变量**。

`编译时`块内部的宏作用范围是`编译时`所在代码块，不受内部代码块的限制。

`编译时`块也可以用`[]`捕获代码片段，但不能修改。

可以在`编译时`块或宏内部使用`错误`来产生编译错误。

```mymz
编译时[÷(整数 分母)]{
	如果 分母≠0 跳转 无错误。
	错误“分母不能为0”。
	定义 跳转 无错误。
}
```

错误可以是任意类型，不一定是字符串。

## 外部函数调用

没有名字可以使用**C接口**和其它语言相互调用。

没有名字调用C++：
```cpp
#include<iostream>
extern "C"{
	void output(int n){
		std::cout<<n;
	}
}
```

```mymz
定义 函数[整数] 常量 输出:=外部 “output” 的 output。
输出(0)。//使用std::cout
//也可以这样：
外部 “output” 的 output(0)。
```

C++调用没有名字：

```mymz
定义 外部 C 函数[整数，整数。返回 整数] plus:
=函数[整数 甲，整数 乙]{返回 甲+乙。}
```

定义外部函数时**只能使用C的类型**。

```cpp
extern "C"{
	int plus(int,int);
}
int main(){
	std::cout<<plus(1,1);
	return 0;
}
```

# 细节

## 类型值

`类型` 类型的变量用于存储类型信息。类型值在内存中占用8字节，编码规则如下：

|类型|最高4位|
|---|---|
|`数组`|0000|
|`数组(直接存储)`|0001|
|`组合`|0010|
|`组合(直接存储)`|0011|
|`函数`|0100|
|`函数(直接存储)`|0101|
|`整数`|0110|
|`浮点数`|0111|
|`字符`|1000|
|`指针`|1001|
|`类型`|1010|
|`字节`|1011|
|`字节[2]`|1100|
|`字节[4]`|1101|
|`字节[8]`|1110|
|无|1111|

基本类型只占4位，其余位的值不影响。

长度为**2、4、8**的字节数组也是基本类型。可以把任意类型的值强制转换为字节。

### 类型描述

对于其它复合类型，将最高位改为全0再解引用可以得到类型描述。

#### 数组

数组的类型描述占12字节，前8字节表示类型，后8字节是长度。

**基本类型**的数组可以**直接存储**完整描述：第2字节为类型，后4字节为长度。

#### 组合

组合类型描述的大小是**可变**的。前8字节表示成员数量，后面每8字节一个成员类型。

一个类型编码占4位。类型变量有8字节，所以**少于14个成员且成员都为基本类型**的组合可以**直接存储**：第5~8位为成员数量，接下来每4位表示一个成员的类型。

#### 函数

函数类型描述的大小也是**可变**的。前8字节表示返回类型，之后8字节表示参数数量，再之后每8字节一个参数类型。

返回类型和参数类型都是基本类型且参数数量小于等于14的函数**直接存储**：第5~8位为返回类型，然后4位参数数量，接下来每4位一个参数类型。
