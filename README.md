# 没有名字(mymz)编程语言
## 基本语法
在没有名字语言中，字符分为3种：
- **字**：Unicode中属于字母或数字的字符，以及下划线“\_”。
- **符号**：除字和空白以外的字符
- **空白**：Unicode中的所有空白字符

连续的字是**一个词元**；每个符号是**单独的词元**；空白被**忽略**。  

使用`。`结束一条语句。  
`{}`表示代码块，用来管理作用域。  
`//`后到换行符之间的内容是**注释**，不会被执行。  
`/*`和`*/`之间也是**注释**，可跨行。  
## 变量和常量
```mymz
定义 整数 变量 甲。
定义 整数 常量 乙:=0。
```
用`定义`定义一个变量或常量。如果要定义多个变量，将它们用`，`隔开。  
声明变量时后面可以直接加`:=`进行初始化。如果这么做，可以不指定类型（会自动改为和值相同的类型）。  
常量**必须**在声明时初始化。
```mymz
定义 整数 变量 甲，常量 乙，字符 丙，浮点数 变量 丁。
/*
甲：整数变量
乙：整数常量
丙：字符常量
丁：浮点数变量
*/
```
在类型后加`[元素个数]`将类型改为**数组**。**数组**可以同时存储多个类型相同的元素。数组使用`名称[索引]`访问元素。**索引从0开始**。  
使用`的 长度`获取长度。它的类型是**字节\[4\]**。  
```mymz
定义 整数[10] 变量 数组。
定义 字节[4] 长度:=数组 的 长度。//10
```  
### 字面量
#### 数
和数学中的写法一样：  
```mymz
定义 整数 常量 零:=0。
定义 浮点数 常量 π:=3.1416。
```
#### 字符和字符串
字符串是**字符数组**。  
非ASCII字符只能用字符数组存储，需要占多个字符。  
字符用`‘’`，字符串用`“”`。  
字符串可以跨行，`“`、`”`和`\`需要转义。  
```mymz
定义 字符[9] 变量 字符串:=“a string”。
```
### 作用域
变量的**作用域**为**所在代码块**，离开时将会被**自动删除**。  
允许声明同名变量，但必须**不在同一作用域中**。如果声明名称相同的变量，之前的同名变量将会**隐藏**，直到新声明的变量离开作用域被删除后才重新可见。
## 指针和引用
### 指针
在类型后加`指针`声明一个**指针**，它可以指向对应类型的元素。使用`&`获取变量的**地址**。指针可以指向指针。用`*`解引用，获取指针指向的引用。  
```mymz
定义 整数 变量 值。
定义 整数 变量 指针 变量 甲=&值。
定义 整数 变量 指针 常量 乙=&值。
定义 整数 常量 指针 变量 丙=&值。
```
甲的类型是指向整数变量的指针变量，既可以被修改，也可以修改指向的值；乙不能被修改，但可以修改指向的值；丙可以被修改，但不能修改指向的值。  
指向类型为变量的指针可以指向常量，但反过来不行。  
可以使用`分配`关键字分配存储空间。  
```mymz
定义 整数 变量 指针 变量 甲:=分配 整数。
*甲:=0。
*甲:=1。
释放 甲。
```
不要释放不是用`分配`分配的存储空间，也不要多次释放同一片存储空间。可以对空指针使用释放。  
**当然，也不要不释放空间。**  
可以但**不推荐**使用无类型指针。  
### 引用
```mymz
定义 整数 变量 甲。
定义 整数 变量 引用 乙:=甲。
```
这会声明一个**引用**，甲和乙完全相同。
乙被修改时甲的值也会变化。  
```mymz
定义 常量 引用 乙:=甲。
```
这次不能用乙修改甲。  
引用变量的作用域比源变量大时会导致**悬垂引用**，这是**未定义行为**。  
### 停用名称和移动语义
使用`停用`停止使用一个名称。  
```mymz
定义 整数 变量 甲。
甲:=0。
定义 整数 变量 引用 甲的引用。
停用 甲。
定义 浮点数 变量 甲。
甲:=0.1。//甲是浮点数变量
甲的引用:=1。
```
停用不影响作用域，使用引用仍然可以修改原变量，不会导致悬垂引用。  
当变量被赋值后值源变量就失效时，将会触发移动语义。  
```mymz
定义 整数[1000] 变量 很大。
定义 整数[1000] 变量 移动。
移动:=很大。
停用 很大。
{
	定义 整数[1000] 变量 局部。
	移动:=局部。//也会触发移动
}
```
## 复合类型
```mymz
定义 类型 常量 二维向量:=组合[浮点数 x，浮点数 y]。  
二维向量 A。
A 的 x:=0。
A 的 y:=0。
```
类型也是值，可以用常量存储。用`的 类型`获取类型。  
**类型不存储可变性。**  
用`组合`定义的类型可以**同时存储所有成员**。  
可以用`类型`类型的变量作为类型声明其它变量。  
使用未初始化的类型变量声明其它变量是**未定义行为**。  
用`的 大小`获取类型的大小。  
## 程序流控制
只有跳转和条件跳转。  
### 跳转
用`定义 跳转 名称`声明一个**跳转点**。  
`跳转 名称`将**跳转**到那里。  
```mymz
定义 字节[4] 变量 计数器:=0。
定义 跳转 循环开始。
//……
计数器:=计数器+1。
如果 计数器<10 跳转 循环开始。
```
### 条件跳转
```mymz
定义 字节 变量 条件。
条件:=0。
如果 条件 跳转 条件满足。
//……
定义 跳转 条件满足。
```
## 运算
`:=`：赋值(前面已经出现过很多次了)  
`()`在括号中的将会**优先**计算  
`+-×÷`：数学运算，加减(相反数)乘除。  
`mod`：取余数  
`=≠＞＜≥ ≤`：比较  
`& | ~ ^ << >>`：位运算  
## 函数
函数也是值。  
```mymz
定义 函数[整数，整数。返回 整数] 常量 加:
=函数[整数 甲，整数 乙。返回 整数]{
	返回 甲+乙。
}
定义 整数 常量 一加一的和:=和(1，1)。
```
`[]`中是**参数**和**返回类型**，后面中是函数要执行的语句。返回类型不写就是无返回值。  
使用`返回`关键字让函数返回一个值。类型不符合时会**自动强制转换**。  
返回导致**临时变量失效**也可以触发**移动语义**。  
传入的参数时将使用**值传递**，修改不影响原变量。如果要在函数中修改变量的值，需要使用引用。  
变量可以被传给接受常量引用的函数，常量不能传给接受变量引用的函数。  
函数被传递到内部变量失效的地方并调用会导致**悬垂引用**，同样是**未定义行为**。  
## 命名空间
命名空间可以隔离名称，你可以在不同的命名空间里使用相同的名称而不会引起歧义。  
用`外部：`在命名空间中指定使用外部的名称。命名空间的名称不能是`外部`。  
```mymz
定义 命名空间 示例命名空间。
定义 整数 变量 甲。
命名空间 示例命名空间{
	定义 整数 变量 甲:=0，乙。
}
甲:=示例命名空间：甲。 //将外部的甲赋值为内部的甲
使用 命名空间 示例命名空间。//使用整个命名空间
乙:=甲。//这是示例命名空间内的甲
外部：甲:=甲+1。//外部的甲
停用 示例命名空间。
甲:=1。//外部的甲
使用 示例命名空间：乙。//使用示例命名空间内的乙
乙:=甲+1。
```
命名空间可以嵌套。  
## 外部函数调用
没有名字可以使用**C接口**和其它语言相互调用。  
没有名字调用C++：
```cpp
#include<iostream>
extern "C"{
	void output(int n){
		std::cout<<n;
	}
}
```
```mymz
定义 函数[整数] 常量 输出:=外部 “output” 的 output。
输出(0)。//使用std::cout
//也可以这样：
外部 “output” 的 output(0)。
```
C++调用没有名字：  
```mymz
定义 外部 C 函数[整数，整数。返回 整数] plus:
=函数[整数 甲，整数 乙]{返回 甲+乙。}
```
定义外部函数时**只能使用C的类型**。  
```cpp
extern "C"{
	int plus(int,int);
}
int main(){
	std::cout<<plus(1,1);
	return 0;
}
```
# 预处理
注意到没有名字的**控制流只有跳转**，代码非常难读，所以……  
用`⟦⟧`标记代码在预处理时运行。当`⟦⟧`中有内容时将会匹配代码片段并在匹配到时运行，否则只运行一次。
## 增加的类型
### 匹配器
匹配器用`⟦⟧`表示。它会根据里面的内容查找符合条件的标志符。找不到时结果为`空`。  
使用`⦅类型 名称⦆`捕获变量。其中类型可以是`标志符`，将匹配任意标志符。  
不能匹配引号`“”`和注释符号`// /**/`。  
### 代码片段
代码片段用`⦃⦄`表示。可以在后面后加`：`指定捕获变量的来源。来源可以有多个，之间用`、`隔开。表示`⦃⦄⟦⟧⦅⦆`时需要转义。  
`开头的`匹配代码片段**开头往后的内容**。  
`开头后面的`**从代码片段的开头向后查找**。  
`结尾的`和`结尾前面的`同理。  
如果某个代码片段是其它代码片段中某一部分的引用，还可以用匹配器向前和向后查找。  
`前面的`匹配该片段**前面的内容**。  
`之前的`**从代码片段的前一个词元开始向前查找**符合条件的代码片段。  
`后面的`和`之后的`同理。  
`之间的`获取两个代码片段之间的代码。
用`+`**连接**多个代码片段。  
```mymz
定义 代码片段 变量 代码:=⦃1 甲 2 乙 3 丙 4⦄。
定义 代码片段 变量 引用 乙:=代码 的 ⟦乙⟧。

代码 开头的 ⟦⦅整数⦆⟧ //⦃1⦄
代码 开头的 ⟦⦅标志符⦆⟧ //空
代码 开头后面的 ⟦⦅标志符⦆⟧ //⦃甲⦄
代码 结尾的 ⟦⦅整数⦆⟧ //⦃4⦄
代码 结尾前面的 ⟦⦅标志符⦆⟧ //⦃丙⦄
乙 前面的 ⟦⦅整数⦆⟧ //⦃2⦄
乙 前面的 ⟦⦅标志符⦆⟧ //空
乙 之前的 ⟦⦅标志符⦆⟧ //⦃甲⦄
乙 之后的 ⟦⦅标志符⦆⟧ //⦃丙⦄

定义 代码片段 变量 引用 甲:=代码 的 ⟦甲⟧。
定义 代码片段 变量 引用 丙:=代码 的 ⟦丙⟧。

甲、丙 之间的 //⦃2 乙 3⦄
甲+丙 //⦃甲 丙⦄
```
## 代码修改
预处理的作用域和变量相同，离开作用域后停止运行。  
可以用`停用 ⟦⟧`**停止**对应匹配器的预处理。  
某段代码同时满足多个匹配器时，会运行后定义的。如果之后出现错误，会回溯并尝试使用另一个匹配器。  
```mymz
定义 类型 常量 二维向量:=组合[浮点数 x，浮点数 y]。
定义 函数[浮点数，浮点数。返回 二维向量] 常量 构造二维向量:
=函数[浮点数 x，浮点数 y。返回 二维向量]{
	定义 二维向量 结果。
	结果 的 x:=x。
	结果 的 y:=y。
	返回 结果。
}
⟦(⦅浮点数 x⦆,⦅浮点数 y⦆)⟧ 二维向量构造函数{
	二维向量构造函数:=⦃构造二维向量(⦅x⦆，⦅y⦆)⦄：二维向量构造函数。
}  
定义 函数[二维向量 常量 引用,二维向量 常量 引用。返回 二维向量] 加:
=函数[二维向量 常量 引用 甲,二维向量 常量 引用 乙。返回 二维向量]{
	定义 二维向量 结果。
	结果 的 x:=甲 的 x+乙 的 x。
	结果 的 y:=甲 的 y+乙 的 y。
	返回 结果。
}
⟦⦅二维向量 甲⦆+⦅二维向量 乙⦆⟧ 二维向量加{
	二维向量加:=⦃加(⦅甲⦆，⦅乙⦆)⦄：二维向量加。
}

定义 二维向量 a:=(1,1)+(2,2)。//可以直接这样
```  

用`定义 标志符`在创建一个**唯一**的标志符。它不会和外部的名称冲突。  
```mymz
⟦循环 ⦅字节[4] 次数⦆ 次 ⦑{} 循环体⦒⟧ 循环{
	循环:=⦃
		定义 标志符 计数器。
		定义 字节[4] 变量 计数器:=0。
		定义 标志符 循环开始。
		定义 跳转 循环开始。
		⦑循环体⦒
		计数器:=计数器+1。
		如果 计数器＜⦅次数⦆ 跳转 循环开始。
	⦄：循环。
}  
定义 整数 变量 甲:=2。
循环 5 次{
	甲×=2。
	甲-=1。
}
```
```mymz
⟦÷⦅整数 分母⦆⟧{
	定义 标志符 无错误。
	如果 ⦅分母⦆≠0 跳转 无错误。
	错误“分母不能为0”。
	定义 跳转 无错误。
}
```
在预处理块内使用`错误`来产生编译错误。  
错误可以是任意类型，不一定是字符串。  
预处理代码只能访问**常量**或**预处理代码中定义的变量**。  
## 多文件
使用`引用`引用另一个文件。文件名是字符串，如果不加`“”`则是标准库。  
文件名必须是编译时能确定的。  
```mymz
引用 “另一个文件”//不需要后缀
引用 语法扩展。//标准库
引用 “文件夹/”。//引用文件夹内的所有文件
引用 容器 的 命名空间 变长数组。//引用指定命名空间
引用 容器 的 变长数组。//引用指定命名空间并提升到外部
```
